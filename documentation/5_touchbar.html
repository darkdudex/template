FAQ

What are these starters I keep seeing?

Each starter represents an empty template to begin with, having the components in the name present.

How do I install the item?

Copy and paste all the items in the code folder to your server to your desired location. 

What are those i class "ion " things?

That is Font Awesome, the amazing icons we used for this item you can learn more about them and find a cheatsheet on how to use them here http://ionicons.com/

How do I change the contact form information?

There is only one PHP file in the item, please open that item and replace the email address with yours. 

How do I make the login page / sign in page / calendar page / blog  to work? 

Please remember that this is a site template and is not connected to any CMS or any databases, so these features are mearly template designs, they are HTML and CSS and need to be connected to a back-end to work. 

I want to add my own JS, it's not working, why?

Add any JS you want starting from line 4. THis is very important since the AJAX refreshes the functions and they are present starting with line 3 and down to 927. When you tap a link that entire zone is refreshed. Be sure to import your script in all pages you want it to work. My suggestion is to copy the min version of your script in plugins.js and import it there as we imported ours. 

This uses AJAX? How to I use a simple link? They don't work! 

Any link that points outside the main item should have the class default-link. This will terminate the smooth state / ajax page loading for that link and it will work like anyother regular link! 

I don't want AJAX. I want the old school loading style. How? 

Send us a message, we'll explain the lines of code that need to be deleted in order to remove this. It's just a small change in custom.js and we'll gladly help you with it.

I'm a tech guru. I want more API's for the AJAX. I need them. 

Everything you need to know to customize the AJAX and more API's can be located on the plugin page over at https://github.com/miguel-perez/smoothState.js

I want to build an app using PhoneGap and Cordova. Can you help me? 

We only provide support for the item as is, meaning a site template. PhoneGap and Cordova's pages state clearly that they work wiht ANY html, css and JS item on the planet. So compatiiblity is not an issue. Furthermore, we use the latest and easiest code techniques to assure there won't be issues. 

Do I need an extended license?

If your app will be paid or have in-app paid features, yes. You do. We will pursue legal action against any app on any application store that uses our code and can't provide a license for an extended license use.

Page Menu

The page is comprised of 2 menus, a left, classic sized one and a right full sized one. Each has a different style of code that is very important you follow. 

<div id="sidebar-left">
    <div class="sidebar-scroll">
        <--LOGO GOES HERE-->
        <div class="sidebar-menu">
            <!--Sidebar Left Content-->
        </div>
    </div>
</div>

<div id="sidebar-right">
    <div class="sidebar-scroll">
        <div class="sidebar-small-menu">
            <!--Sidebar Right Content-->
        </div>
    </div>
</div>
The above code will be the structure of our sidebars. Inisde the left sidebar generate a sidebar logo as follows
 
Sidebar Large Logo
<a href="index.html" class="sidebar-logo sidebar-item"></a>
<div class="sidebar-item sidebar-divider"></div>

You can replace the logo with your own by opening style.css and searching for sidebar-logo. Replace the image there with your own and remember to the keep the width and height half of that of your logo to maintain retina / hd aspect. 

Sidebar Large Simple Menu

<a class="sidebar-item active-item" href="index.html"><i class="ion-ios-star-outline"></i>Welcome<i class="ion-record"></i></a>

The above structure will create a simple menu item. This has an icon, a text, and another icon that will be highlighted by the active-item secondary class.

Sidebar Large Submenu

<a class="sidebar-item" data-sub="sub-1" href="#"><i class="ion-ios-home-outline"></i>Home</a>
<div id="sub-1" class="submenu">
    <a class="sidebar-item" href="index-classic.html">Classic</a>
    <a class="sidebar-item" href="index-round.html">Rounded</a>
</div>
The submenu structure a is a bit different but nevertheless easy to use
 
Line 1 is a simple menu item, but this has the attribute data-sub="sub-1"
Line 2 will have the ID identical to the data sub of the menu item it serves for, therefore the ID will be id="sub-1" Using this method, the JS can target the submenu and deploy it automatically.
Line 3 and 4 are simple submenu items. Comrpised of a class, a link and a text
Line 5 closes the submenu 
 
If you wish to make the menu active by default,
 
line 1 add the secondary class active-item
line 2 add the secondary class active-submenu
line 3 or any other submenu add the secondary class active-item
 
Sidebar Small Menu
 
Inside the sidebar small menu you can add social icons, as follows 
 
<a href="#" class="sidebar-item"><i class="ion-social-facebook-outline"></i><em>Facebook</em></a>
Simply duplicate the code above, and replace your icon and your text with the social icon you wish to use. Please make sure to use the secondary class default-link if you wish to post external links.
 
 
FAQ?
 
Can I make both sidebars the same size?
Yes you can. Open custom.js and locate sidebar_left_width and sidebar_right_width. Make both of these the same size. Let's say for example you wish to have 2 large menus. Then, sidebar_right_width will be the same size as sidebar_left_width.
 
After you've done this, delete your initial sidebar-right ID completely ( the whole div) then copy and paste the entire ID sidebar-left and and rename it to sidebar-right. Now both your sidebars will be the same size. 
 
Can I have different sizes for different pages?
That is not possible unfortunately due to AJAX loading. The custom.js that dictates this would require heavy alterations to support such a feature and would most likely affect page performance. 
 
My social icons are not redirecting. I added my social link and it's not opening. Why?
Be sure to add default-link to the anchor of that social icon as a secondary class.
